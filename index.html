RLCNL

RAIR Lab's Logically Controlled Natural Language

NLG Example

The current version provides a way of specifying how logical functors translate into natural language. The define-compositional-nlg operator lets one specify how functors can combine compositionally in terms of sub functors. More complex methods of combination will be built later. 

    (define-compositional-nlg jack () "jack")

    (define-compositional-nlg mary () "mary")

    (define-compositional-nlg loves (x y) x " loves " y)

    (nlg (parse '(loves jack mary))) ==> "JACK LOVES MARY  "
